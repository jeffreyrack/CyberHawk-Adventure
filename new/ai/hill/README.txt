Please follow the following steps to compile and execute the file
1- Open Eclipse program
2- File-->Import then select Existing Project into Workspace
3- Select archive file then browse for the file and press finish
4- At the left side in the package Explorer 
	--right click on parser then compile for compilation
	--right click on parser then run for running the code
5- To get the source code just press double click on the parser and then src then (default package) then double click on parser.java
